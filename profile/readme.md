## What is Stacker?

Stacker is an open-source project generator that lets you **visually configure** your entire full-stack architecture framework, UI system, authentication, database, API layer, and more — then scaffold it locally with a single command.

```
bunx @stacker-oss/cli <template-id>
```

No more:
- Wasting hours configuring the "perfect" starting point
- Copy-pasting from outdated tutorials
- Decision paralysis before writing a single line of code

Just pick what you need. Ship faster.
